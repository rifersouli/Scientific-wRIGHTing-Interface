A solução de sistemas lineares esparsos de alta ordem está inserido em vários ramos da ciência, como por exemplo a engenharia. Por conseguinte, tem havido um grande esforço para resolver ou apresentar soluções aproximadas de tais sistemas de forma eficiente. --{Neste trabalho combina-se o método da esparsificação recursiva com o algoritmo de Cuthill-McKee para obter uma aproximação esparsa para a inversa de uma classe de matrizes esparsas denominadas matrizes de Hodge.}--
