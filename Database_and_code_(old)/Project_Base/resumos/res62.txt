Neste trabalho, aborda-se o problema de identificação de danos estruturais a partir das frequências naturais. Na formulação do problema de identificação de danos é utilizado o Modelo de Superfı́cie de Reposta (MSR) em substituição a um Modelo de Elementos Finitos (MEF) da estrutura. --{No presente trabalho, a estrutura simulada trata-se de uma viga de Euler-Bernoulli simplesmente apoiada, no qual o problema inverso em questão é definido como um problema de minimização, onde objetivo é minimizar uma função de erro definida a partir dos dados experimentais e de resultados analı́ticos previstos pelo MSR.}-- A utilização do método Evolução Diferencial (ED) no problema de identificação de danos é considerado. --[Frente aos resultados numéricos obtidos, a estratégia adotada mostrou-se capaz de localizar e quantificar osdanos com elevada acurácia.]--
